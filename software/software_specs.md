# <p align="center"> Software specifications </p>
Two modes are at disposal for the software part :<br>
  -  Cloud mode, when the hardware is connected to an Internet network and usable on a computer<br>
  -  Edge mode, when no connection is available and the hardware is to use via Bluetooth connection with a smartphone<br><br>

## <p align="center"> OpenAgri online software (Cloud mode)</p><br>
Software technologies used:<br>
1. Web:<br>
  ●	PostgreSQL database<br>
  ●	PHP framework Laravel<br><br>
2. Mobile App:<br>
  ●	Flutter<br>
  ●	ISAR database<br><br>
3. UHF case:<br>
  ●	JDK for embedded development<br>
  ●	SQL lite<br>

## <p align="center">	OpenAgri mobile App (Edge mode)</p><br> 
This mobile application has been developed to collect UHF Data from Page Up UHF smartcases and to compute locally visits and build daily visits reports.<br> 
<u>Keywords/Tags:</u> RFID UHF, livestock electronic identification, visits to areas of interest<br>
<u>Author/Maintainer:</u> F. Gimbert<br>
<u>Organization:</u> Page Up<br>
<u>Contact Email:</u> fgimbert@pageup.fr<br>
<u>Programming Language:</u> Flutter<br>
<u>Operating System:</u> Android 10+<br>
<u>System architecture:</u> The mobile application is a standalone app (no backoffice nor web site needed). To collect data, it must connect, via bluetooth, to a PageUp UHF box.<br>
